A simple Python game where you guess a random number between 1-100.

## How to Play
1. Run `python guessing_game.py`
2. Enter your guesses
3. Get hints (too high/too low)
4. Guess within 10 attempts to win!

## Features
- Random number generation
- Input validation
- Attempt counter
- Play again option
